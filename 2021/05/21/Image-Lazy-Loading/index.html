原生JavaScript实现图片懒加载

什么是图片懒加载

我们有一个页面，展示图片列表的，图片很多多到当前屏幕展示不下。懒加载就是加载可视区域内的图片。一般图片我们是使用 <img> 标签来实现的，浏览器根据标签上的 src 属性来发起获取图片的请求。我们在未进入到可视区域内，先不给 <img> 标签添加 src 属性，这样动态的控制标签 src 属性就可以完成懒加载了。
