JavaScript对象

定义

JavaScript 中的所有都是对象。对象是一种特殊的数据，拥有属性和方法（即对象成员）。

创建对象的方式

对象字面量

new Object() 构造函数

自定义构造函数 - 对象构造器 - 工厂方法

关键字 new

new 的执行过程:

  1. 在内存中创建一个空对象
  2. 让构造函数中的this执行创建的对象
  3. 执行构造函数，在构造函数中设置属性和方法
  4. 返回当前对象

对象遍历的方法

for…in 循环

Object.keys()

返回对象所有可枚举属性的字符串数组。

Object.values()

返回对象自身的所有可枚举属性值的数组。

Object.getOwnPropertyNames

返回对象的所有自身属性的属性名（包括不可枚举属性但不包括Symbol值作为名称的属性）组成的数组。
