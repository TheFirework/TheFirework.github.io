BFC

BFC 是什么？

块格式化上下文（Block Formatting Context，BFC）,是一个独立的块级渲染区域，只有块级盒子参与。内部元素变化不会影响外部元素。

如何创建 BFC

  • float 的值不为 none
  • overflow 的值为 auto、scroll、hidden (推荐使用)
  • display 的值为 table-cell、table-caption、inline-block
  • position 的值为absolute或fixed
